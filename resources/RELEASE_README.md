# GBC Palette Editor

A macOS tool for editing .pal files used in Game Boy Color ROM hacks.

Made by Krizdingus — krizdingus.com

---

## Opening the app

Double-click **GBC Palette Editor.app** to launch it.

If macOS asks you to confirm before opening, click Open. This happens on
first launch for apps downloaded outside the App Store and should not happen
again after that.

---

## Getting started

Click **Open .pal** to open a palette file, or drag a .pal file directly
onto the window.

**example.pal** is included in this folder. It contains four palette blocks
from a real colorization project. Open it to explore the tool before working
on your own files.

---

## Editing colors

Use the dropdown at the top left to switch between palette blocks.

Click a color swatch to select it, then adjust the R, G, B sliders or type
a hex value directly. The raw byte view on the right updates as you edit.

Click **Save** to write your changes back to the .pal file.

---

## Notes

The Notes area at the bottom saves text per palette block. Notes are stored
in a .notes.json file placed next to your .pal file. If you move the .pal
file, move the notes file alongside it to keep them paired.
